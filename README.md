# NAE:IL (내:일)

소상공인 사장님들을 위한 **AI 기반 매장 운영 및 마케팅 자동화 솔루션**입니다.
복잡한 데이터 분석과 마케팅 기획을 AI가 대신하여 사장님들의 시간과 비용을 절약해 드립니다.

## 🚀 주요 기능

### 💡 AI 마케팅 & 인사이트
- **맞춤형 AI 일일 인사이트**: 오늘의 날씨, 기온, 요일을 종합 분석하여 "비 오는 날 따뜻한 국물 메뉴 할인"과 같이 당일 즉시 실행 가능한 마케팅 액션을 콕 집어 제안합니다.
- **요즘 뜨는 상권 트렌드**: 소상공인 최신 트렌드 뉴스레터 및 상권의 변화를 매일 아침 브리핑해 드립니다.
- **AI 마케팅 카피라이팅**: 신메뉴나 이벤트 내용만 간단히 입력하면, AI가 인스타그램 피드, 당근마켓 단골 알림 등에 최적화된 매력적인 홍보 문구를 3가지 버전으로 자동 작성합니다.

### 📊 매장 운영 & 예측
- **정밀 방문객 예측**: 과거 판매 데이터베이스와 외부 환경 요인(날씨, 요일 등)을 머신러닝으로 분석하여 오늘 예상되는 고객 수를 정확히 예측하고, 식자재 준비 수량을 가이드합니다.
- **스마트 프라이싱 계산기**: 한국농수산식품유통공사(KAMIS)의 실시간 농수산물 도매가 API를 연동하여, 주요 식자재 원가 변동을 반영한 메뉴별 원가율을 분석하고 최적의 판매 가격을 산출해 줍니다.

### 📅 고정비 & 노무/세무 관리
- **스마트 노무 리스크 알림**: 캘린더 기반의 알바생 스케줄러를 제공하며, 주 15시간 초과에 따른 주휴수당 발생 임박이나 초과 근무 등 노무 리스크를 시각적으로 경고해 줍니다.
- **세무 캘린더 체크리스트**: 종합소득세, 원천세, 부가세 등 자영업자가 놓치기 쉬운 세무 신고 일정을 월별로 자동 생성하여 알림을 제공합니다.

### 🤝 커뮤니티
- **동네생활 사장님 게시판**: 지역 인증 기반으로 인근 상권의 사장님들과 익명으로 소통하고, 운영 노하우나 고민을 나눌 수 있는 전용 소통 공간입니다.

## 💻 기술 스택

- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Supabase (PostgreSQL, Edge Functions, Auth)
- **AI Integration**: OpenAI API (GPT-3.5/4) for Insights & Marketing Copy
- **External API**: 
  - KAMIS (한국농수산식품유통공사) 실거래가 오픈 API
  - OpenWeatherMap (일기예보 및 기상 데이터 API)
